# 🛡️ SiberMentor: Otonom Siber Güvenlik Asistanı

SiberMentor, ofansif güvenlik, yazılım zafiyetleri ve otomasyon scriptleri üzerine Sokratik yöntemle eğitim veren, %100 lokal ve Air-Gapped (internetsiz) çalışan bir yapay zeka mentörüdür.

## 🚀 Neden SiberMentor?
* **Operasyonel Güvenlik (OPSEC):** Zafiyetli kodları veya gizli log verilerini 3. parti bulut API'lerine (OpenAI vb.) göndermemek için tamamen lokalde (Llama 3 8B) çalışacak şekilde tasarlandı.
* **Pipeline Mimarisi:** Küçük modellerin yaşadığı halüsinasyon ve kural sızdırma sorunları, "Karar Ağacı" mantığından vazgeçilip katı bir "Boru Hattı (Pipeline)" protokolü kullanılarak çözüldü.
* **Mühendislik Odağı:** Kullanıcıya balık vermek (direkt flag veya exploit kodu vermek) yerine, otomasyon betikleri yazmaya teşvik eden Sokratik bir yönlendirme yapar.
